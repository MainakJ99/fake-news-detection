# Fake News Detection using LSTM

## Overview
This project builds a fake news classification model using LSTM and NLP techniques.

## Dataset
- True.csv
- Fake.csv
Downloaded from Kaggle Fake News Dataset: https://www.kaggle.com/datasets/emineyetm/fake-news-detection-datasets

## Model
- Comparison countvectorizer,tfidf,word2vec,doc2vec models with Logistic regression classifer
- Wordvectoizer+SVM
- Doc2vec+ANN
- LSTM neural network
- Tokenization + Padding

## Files
- lstm_model.keras → trained model
- notebook → training code

## How to Run
1. Preprocess input text
2. Load model
3. Predict
